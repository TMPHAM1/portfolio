    Student Grade Table

    Web Design, Content Management System
    Date : September 2018
    Personal Project Focus: Content Management
    Categories : Apps, CMS
    Website : www.tienmpham.com/sgt

    The Challenge

    The challenge with this project was creating my own database to pull data from and being able to filter the data based on the search terms I defined. There were a lot of issues mostly with displaying and changing information on the client side through passing AJAX.

    The Solution

    Creating the database step by step and creating a restful API. I was able to use CRUD operations (Create, Read, Update, and Delete) to manipulate data. After that I added a sort function which called the database and sorted based on mySQL functions.

    The Result

    A simplistic clean application in which someone can keep records of student data as well as filtering the highest grade or by course or student name.

    Concert Buddy

    Web Application
    Date : September 2018
    Client : Personal Project
    Categories : App, Web Design, UI Kits
    Website : www.concertbuddyapp.com

    The Challenge

    This was an application people could use to plan concerts as well as places to stay, transportation and additional responsibilities for a group of friends. In charge of learning relational databases and planning the back-end functionality. Creating a user login from scractch with the little knowledge known and allowing functionality across the boards.

    The Solution

    I took one step at a time working with my team and communicating what I needed for inputs for the Login. I was able to build the login from scratch as well as adding checks for user authentication. I was able to create a full functioning user system as well as the ability to invite friends and have them create concerts.

    The Result

    A fully functioning application in which each user can view a shared concert with responsibilities and tasks. Users are able to invite friends, leave trips and assign tasks to other people in the group.
