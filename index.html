    Racks Right

    Web Application
    Date : June 2023
    Client : Personal Project
    Categories : App, Web Design, UI Kits
    Website : www.racksright.com

    The Project

    An application that allows the user to select the weight they are trying to lift at the gym for exercises such as sqauts, bench press, or deadlifts. The reason for the application is because different gyms have a variety of weight plates and bars. This allow a person to quickly rereack and be efficient with workout without stressing with what plates they have available. The bar and plates weights are customizable for any case.

    My Main Role

    My main role was to integrate more recent React fundamentals such as hooks into this project. I also designed all of the logos and wanted to create something easy for the user to use quickly. The next goal is to get this application into mobile (iOS first) it will allow me to expand into react native I do want to expand this application into having user login so users can just pull up a program and select what weight they are on so they do not have to manually input.

    My Growth

    I learned how to create my own database schema as well as how mySQL organizes and is able to relate tables. I learned a lot about connecting the backend and front end as I implemented many react components that required calling the server for information. I learned how to cooperate with a group of people in decision making and balancing workloads to make a more robust application, this experience truly taught me the values of working within a team to understand each others strengths and weaknesses.
